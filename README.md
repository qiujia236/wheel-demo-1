# wheel-demo-1

## 安装

### 添加 css

使用本框架前，请在 CSS 中开启 border-box

```css
* {
  box-sizing: border-box;
}
```

你好需要设置颜色变量

```css
:root {
  --button-height: 32px;
  --font-size: 14px;
  --button-bg: white;
  --button-active-bg: #eee;
  --boreder-radius: 4px;
  --color: #333;
  --border-color: #999;
  --border-color-hover: #666;
}
```
